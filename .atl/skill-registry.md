### Nx Guidelines (from AGENTS.md)

- **Task Execution**: Always prefer running tasks through `nx` (i.e., `nx run`, `nx run-many`, `nx affected`) instead of using the underlying tooling directly
- **Workspace Analysis**: Use `nx_workspace` tool to gain understanding of workspace architecture
- **Project Details**: Use `nx_project_details` tool to analyze specific project structure and dependencies
- **Nx Docs**: Use `nx_docs` tool for configuration questions and best practices

### CI Error Guidelines

When fixing CI pipeline errors:

1. Retrieve current CIPEs using `nx_cloud_cipe_details`
2. Use `nx_cloud_fix_cipe_failure` to get logs for specific tasks
3. Fix the problem and verify by running the task
